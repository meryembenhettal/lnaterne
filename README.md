# 💡 Application Lanterne

Cette application "Lanterne" permet aux utilisateurs d’analyser des actualités via plusieurs modes d’entrée (texte, lien, image, audio) et d’obtenir une prédiction . L’application propose aussi un mode sombre/clair pour une meilleure expérience utilisateur.

## Technologies utilisées

- Langage : Python
- Base de données : MariaDB / MySQL  
- Outils : XAMPP
- Interface : FLASK

## Base de données

- Nom : `pfe`  
- Tables principales :  
  - `users` (utilisateurs)  
  - `analyses` (historique des analyses)  

---

## Fonctionnalités principales

- Changement entre mode sombre et mode clair  
- Pages :  
  - À propos  
  - Fonctionnalités  
  - Contact  
- Affichage des dernières actualités via des API externes  
- Plusieurs modes d’entrée pour les analyses :  
  - Texte (actif)  
  - Lien (bientôt activé)  
  - Image (bientôt activé)  
  - Audio (bientôt activé)  
- Système de connexion pour activer tous les modes et accéder à l’historique complet  
- Historique des analyses sauvegardé et affiché à côté  

---



## Installation et utilisation

1. Installer XAMPP ou un serveur MariaDB/MySQL.  
2. Importer la base de données via le fichier `bd_lanterne.sql`.  
3. Configurer les paramètres de connexion à la base dans l’application.  
4. Lancer l’application et utiliser les fonctionnalités décrites.
